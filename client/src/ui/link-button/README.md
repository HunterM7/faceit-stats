# LinkButton

`LinkButton` — ссылка, стилизованная как кнопка.

## Импорт

```tsx
import { LinkButton } from '@/ui/link-button/link-button'
```

## Пропсы

- Поддерживает нативные пропсы `<a>`.
- `disabled?: boolean` — отключает взаимодействие (визуально и функционально).

## Пример

```tsx
<LinkButton href='https://example.com' target='_blank' rel='noreferrer'>
  Открыть
</LinkButton>
```
